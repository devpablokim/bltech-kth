# My PMS — 개인 업무 관리 시스템

브라우저에서 바로 쓰는 가벼운 개인 PMS입니다. 서버·DB·빌드 없이 HTML/CSS/JS 3개 파일로 동작합니다.

## 실행 방법

아무 정적 서버로 이 폴더를 열면 됩니다.

```bash
cd test1
python3 -m http.server 8742
# → http://localhost:8742
```

또는 `index.html`을 브라우저로 직접 열어도 됩니다.

## 구성

| 메뉴 | 설명 |
|------|------|
| 🏠 대시보드 | 오늘 남은 일·진행 중·완료·일정 요약, 카테고리별 진행률 |
| 🔥 오늘 할 일 | 📌 핀을 꽂았거나 마감일이 오늘/지난 작업이 자동으로 모임 |
| 💼 회사 업무 | 칸반 보드 (할 일 → 진행 중 → 완료, 드래그 이동) |
| 🏡 개인 업무 | 칸반 보드 |
| 🌱 자기계발 | 칸반 보드 |
| 📅 개인 일정 | 미니 달력 + 날짜별 일정 등록/조회 |

## 사용 팁

- **빠른 추가**: 각 화면 상단 입력칸에 적고 Enter
- **상세 추가/수정**: 우측 상단 `＋ 새 작업` 또는 작업 제목 클릭 (우선순위·마감일·태그·메모)
- **상태 변경**: 목록에선 상태 칩 클릭으로 순환, 보드에선 카드 드래그
- **오늘 고정**: 📌 버튼 → "오늘 할 일"에 표시
- **검색**: 상단 검색창 (제목·태그·메모 대상)

## 데이터

- 모든 데이터는 **브라우저 localStorage**(`pablo_pms_v1` 키)에 저장됩니다. 서버로 전송되지 않습니다.
- 사이드바 하단 **⬇ 백업 / ⬆ 복원**으로 JSON 파일 백업·이전이 가능합니다.
- 주의: 브라우저(또는 포트)가 바뀌면 데이터 저장소도 달라지므로, 옮길 땐 백업 파일을 사용하세요.
